# Tamper-Resilient QPKBE Implementation

This repository contains the implementation of the proposed **Tamper-Resilient Quantum Public-Key Broadcast Encryption (QPKBE)** scheme.

## Overview

This project implements the proposed Tamper-Resilient QPKBE construction and demonstrates its functionality using Python and Qiskit.

## Requirements

- Python 3.12
- Qiskit 2.5
- NumPy

## Installation

Clone the repository:

```bash
git clone https://github.com/NikitaSrivastav/Tamper-Resilient-QPKBE-Implementation.git
```

Move into the project directory:

```bash
cd Tamper-Resilient-QPKBE-Implementation
```

Install the required packages:

```bash
pip install -r requirements.txt
```

## Run

Execute the implementation:

```bash
python main.py
```
